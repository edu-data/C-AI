# GAIM Lab v3.0 분석 보고서

이 폴더에는 교실 수업 영상 분석 결과 보고서가 포함됩니다.

## 📊 보고서 종류

### 1. 최종 통합 보고서 (HTML)
- **파일**: `최종_통합_보고서.html`
- **내용**: 18개 모든 영상의 종합 분석 결과
- **기능**: 인터랙티브 통계, 차트, 과목별/교사별 비교

### 2. 대시보드 (HTML)
- **파일**: `batch_dashboard.html`
- **내용**: 영상별 점수 분포, 등급 분포, 통계
- **기능**: Chart.js를 활용한 시각화

### 3. 개별 분석 보고서 (마크다운)
- **폴더**: `individual_reports/`
- **파일**: `class_001_상세분석.md` ~ `class_018_상세분석.md`
- **내용**: 각 영상별 상세 분석 및 피드백

### 4. 데이터 파일
- **JSON**: `batch_results.json` - 원본 분석 데이터
- **CSV**: `batch_summary.csv` - 스프레드시트 형식

## 🚀 보고서 생성 방법

```bash
# 배치 분석 (모든 보고서 자동 생성)
python scripts/batch_analysis_18videos.py

# 웹 브라우저에서 보기
# Windows: start reports/최종_통합_보고서.html
# Linux/Mac: open reports/최종_통합_보고서.html
```
